Реализованные решения для промышленных предприятий

Система управления электрическим приводом

Задача:

Разработка системы управления для электрического привода, состоящего из 2-х двигателей мощностью 375 кВт каждый.

Решение:

Создана комплексная система управления на базе ПЛК Siemens с разработкой SCADA-системы.

Результат:

Повышение надёжности оборудования и сокращение времени на обслуживание на 30%.

Стенд для испытаний изделий под нагрузкой

Задача:

Разработка(Замена импортной) системы управления сервоприводами стенда для испытаний с HMI интерфейсом.

Решение:

Создан интерфейс HMI с алгоритмами управления сервоприводами по цифровому протоколу ModBusTCP

Результат:

Замена импортного HMI и системы управления

Сбор парметров работы системы в базу данных

Задача:

Подключиться к ПЛК, считать данные с сокета TCP и передать их в базу данных MySQL

Решение:

Разработано приложение для ПК на языке C++

Результат:

Возможность архивирования данных для дальнейшего анализа и построения графиков

Объединение установок различных производителей

Задача:

Интеграция оборудования разных производителей в единую систему управления.

Решение:

Разработан шлюз для преобразования протоколов, создана единая SCADA-система.

Результат:

Увеличение надёжности и сокращение непроизводственных простоев на 25%.

Конвейерная линия

Задача:

Разработка программного обеспечения и SCADA системы для автоматизации конвейерной линии.

Решение:

Разработана новая система на базе современных ПЛК с интеллектуальным управлением.

Результат:

Увеличение объёмов отгрузки материалов

Система управления насосом

Задача:

Внедрение в общую сеть дополнительного насоса.

Решение:

Разработана система управления насосами и SCADA система.

Результат:

Повышение надёжности
